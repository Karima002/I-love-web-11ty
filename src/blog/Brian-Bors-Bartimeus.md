### Anything with less than 8 limbs
- Alle beperkingen komen voort uit wat mensen kunnen. Dit is waar de wetgeving op gebaseerd is. Als je de website bouwt voor een gebruiker die ziet en niet voor gebrukikers die niet kunnen zien dan ben je aan het discrimineren.

### Het curb cut effect
- Een curve in een stoeprand is ook geschikt voor een kinderwagen.
- Als je een website ontwerpt dan help je daarmee ook andere doelgroepen.
- Waarom is het voor blinde gebruikers belangrijk dat er `h2` is gebruikt? Wij zien de tekst en wij zien dat de tekst eruit ziet dan de tekst eronder. Een blind persoon krijgt de tekst opgelezen en dan mis je de gehele structuur. Dat is een goede reden om het toegankelijk te maken. De doelgroep die met de `h2` ook geholpen wordt is de crawler van Google.
- 80% van de mensen die ondertiteling gebruiken zijn niet de mensen die het nodig hebben maar de mensen die het fijn vinden.

### WCAG: 1.3.1
- Success Criterion (SC)
- Information, structure, and relationships conveyed through presentation can be programmatically determined or are available in text.
- Success Criterion (SC) heeft relatie met de criteria. Software moeten kunnen bewijzen dat de structuur aanwezig is. Ze konden ook zeggen "zorg ervoor dat je teksten en tabelen duidelijk zijn". 
- Deze zit onder percievable
- Deze criteria heeft geen uitzondering

### Waarom je niet alles automatisch kunt toetsen

### 4.1.2 in mensentaal
- 'For all user interface components (including but not limited to: form elements, links and components generated by scripts), the name and role can be programmatically determined; states, properties, and values that can be set by the user can be programmatically set; and notification of changes to these items is available to user agents, including assistive technologies.'
- Bij alle interactieve elementen eist hij de naam en rol (bijv een hamburger menu).
- Naam: wat doet deze knop en waar is hij voor bedoeld? Bijv menu openen.
- Soort: wat is de rol van het element, meestal vanuit het HTML element
- In `div` kan dit met `role=""` maar je zou hiervoor beter het standaard `button` element gebruiken.

### ARIA
- ARIA: set aan extra attributen om deze toegankelijker te maken., Geen ARIA is beter dan ARIA. https://www.w3.org/WAI/ARIA/apg/patterns/
- ARIA is niet bedoeld om aan html toe te voegen maar aan alle XML talen toe te voegen (HTML is een vorm van een XML taal).
- Een three view (bijv van bestanden in Windows) kan je met JS bouwen. Dit zou je ook kunnen doen met aria met `<div role="tree"`. Hiermee zeg je dat deze three view lijkt op de three view zoals je het kent.
